Sistem Catatan Personel & Manajemen Pelanggaran

USER MANUAL V1.1

Selamat datang di Sistem CDS (Catatan Personel). Panduan ini disusun untuk membantu Operator Satker dan Admin Polda dalam mengoperasikan aplikasi secara efektif dan akurat untuk mendukung data kepolisian yang presisi.

Fitur Unggulan

📊 Dashboard Statistik Real-time

Visualisasi data personel dan rekap pelanggaran secara instan dan akurat.

🔍 Pencarian Cerdas & OCR

Mencari data dengan cepat dilengkapi fitur pemindaian nama otomatis dari dokumen.

📝 Sistem Audit Log Lengkap

Setiap aktivitas user tercatat untuk menjaga transparansi dan akuntabilitas data.

📂 Manajemen Dokumen Digital

Unggah dan simpan lampiran berkas (PDF/Gambar) langsung ke catatan personel.

✅ Alur Approval Terintegrasi

Sistem verifikasi data berjenjang untuk menjamin keaslian dan validitas input.

🚀 Optimasi Akses Cepat

Antarmuka yang responsif dengan sinkronisasi data otomatis di latar belakang.

1. Akses dan Keamanan

1.1 Login Ke Sistem

  1. Buka peramban (browser) modern seperti Google Chrome atau Microsoft Edge.
  2. Akses alamat URL aplikasi yang telah diberikan oleh Admin.
  3. Masukkan Alamat Email dan Kata Sandi Anda.
  4. Klik tombol Login.

Penting: Jika Anda lupa password, segera hubungi Admin Polda untuk dilakukan reset akun.

1.2 Keamanan Akun

  • Selalu jaga kerahasiaan kata sandi Anda, jangan berbagi akun dengan personel lain.
  • Gunakan fitur Ganti Password di menu Pengaturan > Profil secara berkala untuk menjaga integritas data.
  • Logout segera setelah selesai menggunakan aplikasi, terutama jika menggunakan perangkat bersama atau komputer kantor.

2. Dashboard (Beranda)

Halaman utama memberikan ringkasan data personil secara real-time melalui visualisasi kartu dan tabel rekap.

  • Kartu Statistik: Menampilkan jumlah total personil, catatan aktif, penanganan hukuman, hingga jumlah data yang membutuhkan persetujuan (Approval) dari Admin.
  • Auto-Refresh: Sistem secara cerdas memperbarui data statistik setiap 60 detik tanpa perlu menyegarkan halaman secara manual.
  • Tabel Rekap Satker: Menampilkan perincian kasus per Satuan Kerja. Angka di dalam kolom dapat diklik untuk melihat rincian personil pada kategori tersebut.
  • Live Indicator: Perhatikan titik hijau berkedip di pojok kanan atas dashboard. Jika berkedip, berarti data Anda sedang sinkron dengan server pusat.

3. Manajemen Data Personel

Menu ini digunakan untuk mengelola basis data induk anggota Polri dan PNS di lingkungan Polda Jabar.

3.1 Menambah Personel Baru

  1. Navigasi ke menu Personel.
  2. Klik tombol Tambah Personel.
  3. Isi formulir dengan lengkap. Pastikan NRP/NIP sesuai dengan identitas asli karena sistem akan melakukan validasi keunikan data.
  4. Klik Simpan.

3.2 Menonaktifkan / Menghapus Personel

Sistem CDS tidak menghapus data secara permanen demi kepentingan audit sejarah. Untuk menonaktifkan data:

  1. Klik ikon Tempat Sampah pada baris personel.
  2. Pilih alasan status data (Meninggal Dunia, Pensiun, atau Kesalahan Input).
  3. Wajib mengisi kolom alasan sebagai dasar arsip digital.
  4. Data akan berpindah secara otomatis ke kategori Tidak Aktif.

4. Catatan Pelanggaran

Halaman ini adalah inti dari sistem CDS, berfungsi sebagai buku catatan riwayat pelanggaran personil.

4.1 Menambah Catatan Baru

  1. Gunakan kolom pencarian untuk menemukan personel yang dimaksud.
  2. Klik tombol Action > Tambah Pelanggaran.
  3. Isi detail lengkap termasuk wujud perbuatan, nomor surat dasar, dan tanggal kejadian.
  4. Lampiran: Unggah berkas pendukung (PDF/Gambar) dengan ukuran maksimal 5MB per file.
  5. Klik Simpan. Catatan akan berstatus Draft hingga diverifikasi oleh Admin.

5. Administrasi & Konfigurasi

Fitur ini hanya dapat diakses oleh pengguna dengan level akses Admin Polda atau Superadmin.

  • Manajemen Satker: Untuk menambahkan unit kerja baru atau melakukan perubahan nama Satker.
  • Audit Log: Rekam jejak seluruh aktifitas user di dalam sistem (siapa, melakukan apa, kapan, dan alasannya).
  • User Management: Pengelolaan hak akses akun operator di seluruh Satuan Wilayah dan Satuan Kerja.
