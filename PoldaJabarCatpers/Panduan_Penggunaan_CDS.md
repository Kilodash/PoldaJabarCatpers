---

## 1. Akses dan Keamanan
### 1.1 Login Ke Sistem
1. Buka peramban (browser) dan akses alamat aplikasi.
2. Masukkan **Alamat Email** dan **Kata Sandi** yang telah didaftarkan.
3. Klik tombol **Login**.

### 1.2 Keamanan Akun
- Selalu jaga kerahasiaan kata sandi Anda.
- Gunakan fitur **Ganti Password** di menu **Pengaturan > Profil** secara berkala.
- **Logout** segera setelah selesai menggunakan aplikasi, terutama jika menggunakan perangkat bersama.

---

## 2. Dashboard (Beranda)
Halaman utama memberikan ringkasan data personil secara real-time.
- **Kartu Statistik**: Menampilkan jumlah total personil, catatan aktif, penanganan hukuman, hingga jumlah data yang membutuhkan persetujuan (Approval).
- **Auto-Refresh**: Data pada dashboard diperbarui secara otomatis setiap 60 detik.
- **Tabel Rekap Satker**: Menampilkan perincian kasus per Satuan Kerja. Anda dapat mengklik angka di kolom untuk melihat rincian personil pada kategori tersebut.
- **Live Indicator**: Titik hijau berkedip menandakan koneksi ke server sedang sinkron dan data adalah yang terbaru.

---

## 3. Manajemen Data Personel
Menu ini digunakan untuk mengelola basis data anggota Polri dan PNS.
### 3.1 Menambah Personel Baru
1. Klik tombol **Tambah Personel**.
2. Isi formulir dengan lengkap. **NRP/NIP** harus unik dan valid.
3. Klik **Simpan**.

### 3.2 Mengubah Data
1. Cari nama personil menggunakan kolom pencarian.
2. Klik ikon **Pensil (Edit)** pada baris nama yang bersangkutan.
3. Sesuaikan data dan simpan kembali.

### 3.3 Menonaktifkan / Menghapus Personel
Sistem menggunakan metode *Audit Log* untuk keamanan data. Untuk menghapus/menonaktifkan:
1. Klik ikon **Tempat Sampah (Hapus)**.
2. **Pilih Status Data**: (Meninggal Dunia / Pensiun / Dihapus karena Koreksi).
3. **Wajib Isi Alasan**: Tuliskan alasan penghapusan secara jelas (arsip audit).
4. Data yang dinonaktifkan akan berpindah ke kategori **Tidak Aktif**.

---

## 4. Catatan Pelanggaran
Menu ini berfungsi sebagai buku catatan riwayat indikasi maupun putusan pelanggaran personil.
### 4.1 Menambah Catatan Baru
1. Cari personil di tabel Pelanggaran.
2. Klik tombol **Catatan (Tambah)**.
3. Isi detail pelanggaran (Wujud perbuatan, tanggal LHP, dasar hukum).
4. **Lampiran**: Anda dapat melampirkan berkas PDF atau Gambar pendukung.
5. Klik **Kirim sebagai Draft**.

### 4.2 Alur Persetujuan (Approval)
- **Draft**: Catatan baru berstatus "Draft" dan belum dihitung sebagai statistik final.
- **Admin Review**: Admin Polda akan meninjau draft tersebut.
- **Disetujui**: Catatan masuk ke riwayat resmi.
- **Ditolak/Revisi**: Admin akan mengembalikan draft jika ada kesalahan dengan menyematkan **Catatan Revisi**. Operator harus memperbaiki data sesuai instruksi tersebut.

---

## 5. Fitur Pencarian & OCR
Aplikasi dilengkapi fitur pencarian cerdas untuk mempercepat penemuan data.
- **Pencarian Teks**: Cari berdasarkan Nama, NRP, atau Jabatan.
- **Pencarian Dokumen (OCR)**: Anda dapat mengunggah file hasil scan (Dokumen) ke menu Pencarian Dokumen untuk mendeteksi nama personil yang terkandung di dalam dokumen secara otomatis (fitur dalam pengembangan/bertahap).

---

## 6. Administrasi & Konfigurasi (Admin Polda Only)
Menu ini khusus untuk pengelolaan infrastruktur aplikasi oleh Admin Polda.
- **Manajemen Satker**: Menambah atau mengurutkan Unit Kerja.
- **Audit Log**: Melihat riwayat aksi modifikasi data secara mendetail.
- **User Management**: Mengelola akun operator di tiap-tiap Satker.

---

## 7. Pengaturan & Keamanan Mandiri (Semua User)
Kini **Operator Satker** dapat mengakses menu Pengaturan untuk keperluan mandiri:
- **Profil**: Melihat rincian akun Anda sendiri (Email, Role, Satker).
- **Ubah Password**: Disarankan untuk mengubah password secara berkala.
  1. Masuk ke menu **Pengaturan**.
  2. Klik tab **Profil & Keamanan**.
  3. Masukkan **Password Saat Ini**.
  4. Masukkan **Password Baru** (min. 6 karakter).
  5. Klik **Perbarui Password Akun**.
